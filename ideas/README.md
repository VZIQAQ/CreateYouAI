# Ideas / 想法

你的想法，大神来实现。

提出你想让 AI 做什么，社区里的开发者会把它变成可用的 Skill。

## 怎么提想法

1. 点 New Issue
2. 标题写：`[Idea] <你想要的 AI 能力>`
3. 正文填写模板（见下方）
4. 社区开发者认领后，会创建 Skill 并提交 PR

## 怎么认领想法

1. 在 Issue 下评论 "I'll take this"
2. Fork 仓库，写 Skill
3. 提交 PR，关联 Issue（`Closes #<issue-number>`）

## 想法模板

```markdown
## 我想要什么 AI

<一句话描述>

## 使用场景

<什么情况下会用到>

## 期望效果

<最好是什么样的>

## 参考

<有没有类似的工具/产品/灵感>
```
